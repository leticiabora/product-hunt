## Getting Started

First, install the packages:

```bash
npm install
```

Create and .env file:

```bash
DEV_TOKEN=your-product-hunt-dev-token
```

Then, start the server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
